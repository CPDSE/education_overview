# SDU Pharmacy Programme — Course Overview

Interactive course overview for the Bachelor of Pharmacy programme at the University of Southern Denmark (SDU), showing the study path from autumn 2024 with integrated data science elements across all semesters.

**Live page:** `https://<your-username>.github.io/sdu-pharmacy-program/`

## What it shows

- All 6 semesters with courses laid out in a grid
- Data science elements per course (expandable)
- CPDSE colour scheme
- Print-ready layout (fits 2× A4 landscape)

## Developed by

[Center for Pharmaceutical Data Science Education (CPDSE)](https://www.cpdse.dk)
University of Copenhagen · University of Southern Denmark

## How to deploy

1. Create a new GitHub repository (public)
2. Upload `index.html` to the root
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your page will be live at `https://<username>.github.io/<repo-name>/`
